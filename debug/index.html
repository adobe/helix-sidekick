Helix Sidekick | Debug Server

Extension

  1. Richt-click extension
  2. Click "Options"
  3. Click "Advanced"
  4. Tick "Test sidekick locally" checkbox
  5. Navigate to a Helix page and invoke Sidekick extension

Bookmarklet

  1. Right-click bookmarklet
  2. Click "Edit"
  3. Change URL to load 
    http://localhost:3001/src/sidekick/app.js
    instead of 
    https://wwww.hlx.live/tools/sidekick/app.js
  4. Click "Save"
  5. Navigate to a Helix page and invoke Sidekick bookmarklet
